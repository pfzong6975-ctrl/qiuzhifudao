# AI 求职辅导平台 — 项目总结报告

## 一、竞品分析与业务背景

### 1.1 业务背景

当前求职市场呈现供需错配态势：2025年全国高校毕业生超 1200 万，但企业普遍反映"招人难"。核心矛盾在于求职者缺乏专业的简历优化能力和面试技巧，而市面上的职业辅导服务价格高昂（单次简历修改 300-800 元，模拟面试 200-500 元/次），大量学生和职场新人难以负担。

疫情后远程面试成为主流，催生了线上面试练习的需求。AI 大模型技术的成熟，使得低成本、大规模、个性化的求职辅导成为可能。

### 1.2 竞品能力与现状分析

| 竞品 | 类型 | 核心能力 | 短板 |
|------|------|----------|------|
| **超级简历** | 简历优化 | 智能简历模板、ATS评分 | 仅限简历，无面试辅导 |
| **牛客网** | 题库+社区 | 海量面经、在线编程 | AI能力弱，无个性化辅导 |
| **脉脉** | 职场社交 | 人脉、内推、公司点评 | 非辅导工具，信息过载 |
| **BOSS直聘** | 招聘平台 | AI职位推荐、即时沟通 | 偏招聘流程，无求职辅导 |
| **VMock** | 海外简历 | AI简历评分、措辞建议 | 仅英文、价格高 |
| **Interview Warmup (Google)** | 面试练习 | AI面试提问、实时转写 | 仅限 Google 面试场景 |

**市场空白**：国内缺少一个覆盖「职业规划→简历→面试→复盘→Offer决策」全流程的 AI 求职辅导平台。现有产品要么只做单一环节（简历或题库），要么 AI 能力薄弱（固定题库、无个性化反馈）。

### 1.3 差异化定位

本平台采用"漏斗式"设计，覆盖求职全链路：

```
职业规划 → 简历制作 → 岗位匹配 → 面试练习 → 复盘提升 → Offer决策
    ↓           ↓          ↓          ↓          ↓         ↓
 性格测试    AI对话写    AI对比     AI面试官   五维雷达   多维度
 +AI评估    简历+优化    JD分析    即时评分    分析报告   对比分析
```

核心壁垒：全流程数据闭环。用户的简历、面试记录、复盘报告形成数据资产，AI 可以基于历史数据提供越来越精准的个性化建议。

## 二、PRD 撰写与 Vibe Coding 开发

### 2.1 PRD 流程

从零撰写完整产品需求文档，包含：
- 产品定位与目标用户画像
- 13 个功能模块的详细定义（输入→处理→输出→交互流程）
- 用户旅程地图（迷茫期→准备期→投递期→面试期→决策期）
- 技术架构选型（前后端分离、AI 双引擎、暗色/浅色主题）
- P0/P1/P2 优先级规划
